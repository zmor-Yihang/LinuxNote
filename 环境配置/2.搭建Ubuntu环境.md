# 3.共享文件夹

## 3.1 VMware设置共享文件夹

<img src="C:\Users\30495\Desktop\Linux笔记\sources\QQ20251030-134625.png" style="zoom:50%;" />

## 3.2 挂载共享文件夹

```bash
sudo mount -t fuse.vmhgfs-fuse .host:/ /home/zmor/share -o allow_other
```

## 3.3 开机自动挂载

使用文本编辑器打开 `/etc/fstab` 文件：

```bash
sudo vim /etc/fstab
```

在文件末尾添加以下行：

```txt
.host:/ /home/zmor/share fuse.vmhgfs-fuse defaults,allow_other 0 0
```

# 4.编译工具安装

基础工具

````bash
sudo apt-get install build-essential
sudo apt-get install git
sudo apt-get install make
sudo apt-get install gcc
sudo apt-get install g++
````

交叉编译工具链安装命令：

```bash
sudo apt-get install gcc-arm-linux-gnueabihf
sudo apt-get install binutils
sudo apt-get install libc6-dev
```

内核和 U-Boot 构建工具

```bash
sudo apt-get install bc
sudo apt-get install bison flex
sudo apt-get install libssl-dev
```

文件系统构建工具

```bash
sudo apt-get install gzip bzip2
sudo apt-get install cpio
sudo apt-get install rsync
```

调试工具

```bash
sudo apt-get install gdb
sudo apt-get install strace
sudo apt-get install lsof
```

其他实用工具

```bash
sudo apt-get install wget
sudo apt-get install unzip zip
sudo apt-get install pkg-config
```
